### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 超市商品
5. 动物信息
6. 车位信息
7. 便民服务
8. 房屋信息
9. 物业人员
10. 社区公告
11. 后台管理
12. 个人中心
13. 我的收藏


**管理员功能：**
1. 登录
2. 系统首页（动物种类统计、车位统计、销售额统计）
3. 个人中心（修改密码、个人信息）
4. 用户管理
5. 商品分类管理
6. 超市商品管理
7. 动物种类管理
8. 动物信息管理
9. 车位信息管理
10. 服务类型管理
11. 便民服务管理
12. 缴费信息管理
13. 房屋类型管理
14. 房屋信息管理
15. 物业人员管理
16. 订单信息管理
17. 反馈意见管理
18. 系统管理（系统简介、轮播图管理、社区公告、关于我们）
